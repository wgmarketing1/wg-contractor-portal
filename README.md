# W&G Contractor Payment Portal

React-based payment portal for managing contractor payments and documents.

## ⚠️ IMPORTANT FIRST STEP

**Before deploying, you MUST replace `src/PaymentPortal.jsx` with your complete component code!**

The current file is just a placeholder. Replace it with your full 1800+ line PaymentPortal component.

## Quick Deploy to Vercel

1. Make sure you've replaced `src/PaymentPortal.jsx` with your full code
2. Push all files to GitHub
3. Go to [vercel.com](https://vercel.com) and sign up
4. Click "New Project" and import this repository
5. Vercel will auto-detect Vite and deploy
6. Get your live URL!

## Embed in Wix

```html
<iframe 
  src="https://your-vercel-url.vercel.app" 
  width="100%" 
  height="900px" 
  frameborder="0">
</iframe>
```
